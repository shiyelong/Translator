文本翻译接口

URL: http://零距离.top/translate_text

请求方法: POST

请求体示例 (JSON):

{
    "text": "Hello, World!",
    "dest": "zh-CN",
    "src": "en"
}

返回示例 (JSON):

{
    "translated_text": "你好，世界！"
}

试用链接:

curl -X POST http://127.0.0.1:5000/translate_text -H "Content-Type: application/json" -d '{"text": "Hello, World!", "dest": "zh-CN", "src": "en"}'

图片翻译接口

URL: http://零距离.top/translate_images

请求方法: POST

请求体示例 (JSON):

{
    "image_urls": ["https://i.imgur.com/3IwV34Q.png", "https://i.imgur.com/4IwV34Q.png"],
    "dest": "zh-CN"
}

返回示例 (JSON):

{
    "translated_texts": [
        "int main() {\nthread_pool pool(2);\nauto worker =\nasync(pool, [] {\nfor(size_t i = 0; i < 10; i++) {\nasync(pool, [] {\nthread_local int count = 0;\n\nstd::this_thread::sleep_for(1s);\nstd::cout\n\n<< \"工作线程:\" << std::this_thread::get_id()\n<< \"\tcount:\" << count++ << std::endl;\n\nstd::this_thread::sleep_for(5ms);\n\n});\n}\n});\n\nreturn worker.get();\n}",
        "线程ID是16064\n睡眠num是100\n线程ID是7600"
    ]
}

试用链接:

curl -X POST http://127.0.0.1:5000/translate_images -H "Content-Type: application/json" -d '{"image_urls": ["https://i.imgur.com/3IwV34Q.png", "https://i.imgur.com/4IwV34Q.png"], "dest": "zh-CN"}'
